# Shoei Driving School Website

A responsive website for Shoei Driving School focused on helping people work and live in Japan.

## Features

- Responsive design
- Smooth scrolling
- Contact form
- Mobile-friendly layout

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/12kirantamang/shoei-driving-school-website.git
```

Open `index.html` in your browser.
